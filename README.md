# Project 0

# ENGO 551 - Adv. Topics on Geospatial Technologies


## Pages (HTML)

- **index.html**  
  The homepage with a hero video, a short introduction, and three clickable cards that link to my main sections.

- **about.html**  
  A page about my interest in space. It includes written sections, a list of my favorite things about space, and a slideshow of cool space images.

- **platform.html**  
  A page about my social media content creation. It includes what I post, a back-to-home button, and clickable buttons linking to my social platforms.

- **art.html**  
  A page about art and creativity. It includes a slideshow and a table that compares art I have done vs art I want to do.

## Styling (CSS / SCSS)

- **styles.scss**  
  This is the code responsible for creating all the style of the webpage and making it look nice

- **img/**  
  Contains images used for the entire webpage

- **vids/**  
  Contains the video for the homepage.


## How to Run

1. Open the project folder in VS Code.
2. Right click `index.html` in VS Code and reveal in file explorer
3. Double click the html file in file explorer to reach the homepage (index) of the webpage
